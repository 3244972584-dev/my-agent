# 实验全自动流水线

## 触发条件

用户提供实验数据图片（和可选的指导书），要求一键完成全部处理。

## 流水线流程

```
📷 数据图片 + 📄 指导书（可选）
         │
         ▼
  ┌─────────────────────────┐
  │ Step 1: 数据提取         │  ← experiment-data-assistant-pro
  │   识别图片 → 结构化数据   │     (视觉模型 + Excel生成)
  └──────────┬──────────────┘
             ▼
  ┌─────────────────────────┐
  │ Step 2: 数据验证         │  ← experiment-data-validator
  │   异常值检测 + 单位检查   │     (发现问题时提示用户确认)
  └──────────┬──────────────┘
             ▼
  ┌─────────────────────────┐
  │ Step 3: 曲线拟合         │  ← experiment-curve-fitting
  │   自动选择拟合 + 计算     │     (线性/多项式/指数)
  └──────────┬──────────────┘
             ▼
  ┌─────────────────────────┐
  │ Step 4: 生成图表         │  ← experiment-plotter
  │   散点图 + 拟合线         │     (matplotlib, 保存图片)
  └──────────┬──────────────┘
             ▼
  ┌─────────────────────────┐
  │ Step 5: 生成报告         │  ← experiment-report-generator
  │   完整Word实验报告        │     (python-docx, 含图表)
  └─────────────────────────┘
```

### Step 1: 数据提取

**调用 skill:** `experiment-data-assistant-pro`

- 用视觉模型（`zai/glm-5v-turbo`）识别图片中的表格数据
- 如视觉模型超时，压缩图片后重试，或换 `zai/glm-4.6v`
- 生成 Excel 文件（openpyxl）
- 保存到桌面

**输出：** 结构化数据 + Excel 文件

### Step 2: 数据验证

**调用 skill:** `experiment-data-validator`

对提取的数据进行质量检查：
- 异常值检测（Z-score / IQR 法）
- 单位与数量级检查
- 趋势合理性分析
- 数据完整性检查

**⚠️ 如发现异常：** 停下来提示用户确认，不要自动修改数据。用户确认后继续。

**输出：** 验证报告

### Step 3: 曲线拟合

**调用 skill:** `experiment-curve-fitting`

- 根据实验类型自动选择拟合方式
- 执行拟合，返回方程、参数、R²
- 评价拟合质量

**输出：** 拟合结果（方程 + 参数 + R²）

### Step 4: 生成图表

**调用 skill:** `experiment-plotter`

- 生成散点图 + 拟合线
- 保存为 PNG 图片
- 如有多组数据，生成对比图

**输出：** 图表图片文件

### Step 5: 生成报告

**调用 skill:** `experiment-report-generator`

整合前面所有步骤的结果，生成完整 Word 报告：
- 实验目的、原理、器材、步骤
- 原始数据表格
- 数据处理过程（含拟合公式）
- 图表（插入 Word）
- 误差分析与总结

**输出：** Word 报告文件

## 最终交付物

| 文件 | 格式 | 内容 |
|------|------|------|
| 实验数据表 | .xlsx | 原始数据 + 处理结果 |
| 实验图表 | .png | 拟合图、关系图 |
| 实验报告 | .docx | 完整报告（含图表） |

所有文件保存到桌面：`/mnt/c/Users/sdg/Desktop/`

## 注意事项

- Step 2 发现异常时必须暂停让用户确认
- 视觉模型超时时自动降级（5v-turbo → 4.6v → 压缩图片重试）
- 每步完成后给用户简要反馈
- 最终结果让用户确认关键数字是否正确
