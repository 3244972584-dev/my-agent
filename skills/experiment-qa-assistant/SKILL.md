# 实验问答助教

## 触发条件

用户提出与实验相关的问题，需要基于实验指导书进行解答。

## 回答流程

### Step 1: 理解问题

分析用户问题的类型：

| 问题类型 | 关键词示例 | 回答重点 |
|---------|-----------|---------|
| 原理类 | 为什么、原理是什么、推导 | 公式 + 物理含义 |
| 步骤类 | 怎么做、操作步骤、怎么测量 | 具体操作 + 注意事项 |
| 数据类 | 怎么算、公式、数据处理 | 公式 + 代入示例 |
| 误差类 | 误差来源、为什么偏差大 | 系统误差 + 随机误差 |
| 仪器类 | 怎么用、仪器原理 | 使用方法 + 规格参数 |

### Step 2: 检索指导书内容

如果指导书已在上下文中，直接引用。如果是 PDF 文件，用 `read` 工具读取相关章节。

### Step 3: 构建回答

**回答规范：**

1. **准确** — 基于指导书原文，不编造
2. **结合原理** — 不仅说"怎么做"，还说"为什么这么做"
3. **避免空话** — 不写"实验中应注意安全"之类无用的话
4. **学生语言** — 用易懂的表达，不用晦涩术语
5. **分点说明** — 复杂内容用编号列表

**回答模板：**

```
❓ 问题：[复述问题]

📖 解答：

[直接回答，1-2句话概括]

[展开说明]

• 要点1
• 要点2
• ...

[如有公式，展示并解释各符号含义]

[如有注意事项，特别标注⚠️]
```

### Step 4: 常见问题快速参考

#### 逸出功实验

**Q: 为什么用里查孙直线法而不是直接测量？**
> A 和 S 无法准确测定。里查孙直线法中，lg(AS) 只影响截距不影响斜率，由斜率即可定出 φ，避免了 A 和 S 不可测的困难。

**Q: 为什么要做肖特基修正？**
> 加速场的存在使阴极表面势垒降低（肖特基效应），测得的 Ie' 大于真实 Ie。必须外推到零场才能得到真实发射电流。

**Q: 温度 T 的测量误差为什么影响很大？**
> 在里查孙公式中，T 出现在指数项 exp(-e₀φ/kT) 中。指数对 T 极其敏感，T 的微小误差会被指数放大。

**Q: 采样电阻 Re 的误差怎么影响结果？**
> Ie' = Ue'/Re，Re 的误差直接传递给 Ie'，再经对数运算传递到 lg(Ie/T²)，最终影响斜率即 φ 值。

## 注意事项

- 回答必须基于指导书，不确定时明确说明
- 涉及公式时标注公式编号（如式(1)、式(2)）
- 如果问题超出指导书范围，给出通用物理知识但标注"指导书未涉及"
- 避免直接给出实验数据的答案，引导用户自己处理
