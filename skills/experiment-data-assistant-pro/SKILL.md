# 智能实验数据助手

## 整体流程

用户通常提供：实验数据图片（纸质记录的照片/截图），可能附带实验指导书文字。

按以下步骤执行：

### Step 1: 解析实验信息

分析图片和用户提供的信息，识别：

1. **实验类型**（电学/热学/化学/力学/光学/其他）
2. **实验名称**
3. **数据字段**及其单位
4. **数据处理方法**

输出结构化信息供后续步骤使用。

### Step 2: 图片数据提取

**关键：必须使用视觉模型（zai/glm-5v-turbo 或 zai/glm-4.6v）识别图片中的表格数据。**

调用 `image` 工具：
- `model`: `zai/glm-5v-turbo`（首选，速度快）或 `zai/glm-4.6v`
- `prompt`: 要求按行列输出所有数据，数字必须精确，用 | 分隔列

提取要点：
- 自动识别表格结构
- 修正常见识别错误（如 0/O, 1/l）
- 保持单位一致
- 将识别结果整理为结构化数据

### Step 3: 生成 Excel

使用 `openpyxl`（已安装）生成 Excel 文件。

参考 [references/excel-template.md](references/excel-template.md) 中的代码模板。

关键要求：
- 保持原始表格格式（合并单元格、表头结构）
- 数字精度与原始数据一致
- 保存到用户桌面（`/mnt/c/Users/sdg/Desktop/`）

### Step 4: 数据分析

根据实验类型推荐分析方法：

| 实验类型 | 推荐分析 |
|---------|---------|
| 电学 | U-I曲线、线性拟合、电阻计算、逸出功 |
| 热学 | 温度变化曲线、热平衡分析 |
| 化学 | 浓度变化、反应速率 |
| 力学 | 位移-时间/速度分析 |
| 光学 | 折射/成像关系 |

如有用户额外指令，优先执行。

### Step 5: 生成实验报告（可选）

用户要求时生成，包含：
1. 标题（实验名称）
2. 实验目的
3. 实验原理
4. 实验数据（表格形式）
5. 数据处理与结果
6. 图表建议
7. 误差分析
8. 实验结论

使用 `python-docx` 生成 Word 文档。如未安装，先 `pip install --user --break-system-packages python-docx`。

## 注意事项

- 图片识别**必须**用视觉模型，不要用 tesseract OCR（对手写/低质量图片效果差）
- 视觉模型可能超时，重试一次；若仍超时，换另一个视觉模型
- 数据提取后让用户确认关键数字是否正确
- Excel 文件保存到 `/mnt/c/Users/sdg/Desktop/`
